# Download & Installasi Docker
## Download Docker

1. Download Installer Docker melalui link [berikut](https://docs.docker.com/get-docker/) dan pilih sesuai sistem operasi yang digunakan.<br>
![gb1](https://github.com/AnggitaAlbiantara/tekn-cloud-computing/blob/8fdf1b8e35602bfa802201d0435272411a18378f/minggu-07/1.PNG)

2. Lalu klik pada button "Docker Desktop for Windows"
![gb2](https://github.com/AnggitaAlbiantara/tekn-cloud-computing/blob/8fdf1b8e35602bfa802201d0435272411a18378f/minggu-07/2.PNG)

3. Tunggu hingga proses download selesai<br>
![gb3](https://github.com/AnggitaAlbiantara/tekn-cloud-computing/blob/8fdf1b8e35602bfa802201d0435272411a18378f/minggu-07/3.PNG)

1. Open installer Docker yang telah di download tadi

2. Centang kedua pilihan seperti gambar dibawah ini<br>
![gb5](https://github.com/AnggitaAlbiantara/tekn-cloud-computing/blob/8fdf1b8e35602bfa802201d0435272411a18378f/minggu-07/5.PNG)

3. Tunggu hingga proses installasi selesai<br>
